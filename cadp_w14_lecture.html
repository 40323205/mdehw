第 14 週開始的規劃

2015Fall 電腦輔助設計實習

呈現零件3D圖(一)-

  1. 把個人所畫的零件存成 stl 檔
  2. 把這個 stl 檔推到個人 github 倉儲後
  3. 在 github 倉儲中點入 stl 檔
  4. 會有一個 Raw 的鈕可以點選

呈現零件3D圖(二)-

  1. 複製網址：從個人帳號開始複製
  2. 運用此語法：script src="https://embed.github.com/view/3d/X" /script
  3. (若想運用此語法，請將 script 及 /script 加上 < >)
  4. 將上一頁第5步驟複製的網址 覆蓋到X
  5. 再將整串放入文章中，即可呈現零件3D圖
